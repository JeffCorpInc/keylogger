# 🔐 Python Keylogger

> ⚠️ **For educational and ethical use only.** Unauthorized use is illegal.

A simple Python keylogger using `pynput` that captures keystrokes and saves them to a log file — built to help understand how keyloggers work for cybersecurity awareness.

---

## ⚙️ Requirements

```bash
pip install pynput
```

---

## ▶️ Usage

```bash
python keylogger.py
```

- Keystrokes are saved to `logfile.txt`
- Special keys (Shift, Enter, etc.) are printed to the console
- Press **Enter** in the terminal to stop

---

## ✨ Features

- Real-time keystroke capture
- Logs standard keys to `logfile.txt`
- Handles special keys without crashing
- Lightweight with minimal dependencies
